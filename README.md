## Crop Recommendation System using Machine Learning

This project is a machine learning-based crop recommendation system that suggests the most suitable crop to cultivate based on environmental and soil conditions. It helps improve agricultural decision-making by analyzing key factors such as nitrogen, phosphorus, potassium levels, temperature, humidity, pH, and rainfall.

The system is built using supervised learning techniques and trained on an agricultural dataset to accurately predict crop types. Multiple models were explored, with a focus on achieving high accuracy and reliable predictions.

### Key Features

* Data preprocessing and encoding of categorical variables
* Model training using algorithms like Random Forest and XGBoost
* Performance evaluation using accuracy metrics
* Prediction system for real-time crop recommendations
* Clean and structured implementation in a Jupyter Notebook

### Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Jupyter Notebook

### Dataset

The dataset contains agricultural parameters such as:

* N, P, K (soil nutrients)
* Temperature
* Humidity
* pH
* Rainfall

### Objective
The main goal of this project is to assist farmers and agricultural planners in selecting the most suitable crop for given environmental conditions, thereby increasing productivity and reducing risk.
